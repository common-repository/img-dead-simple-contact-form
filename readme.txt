== Description ==


Just a simple contact form - for when those bulky forms are too much.

Contact form collects:

* Full name
* Email
* Telephone (optional)
* Message

and emails it to your specified email address and that's it

No CSS - that's up to you!

No management of which form fields to show (except telephone number)



== Installation ==

1. Upload the `img-dead-simple-contact-form` folder to the `/wp-content/plugins/` directory
1. Activate the IMG Dead Simple Contact Form plugin through the 'Plugins' menu in WordPress
1. Enter the shortcode [dscf] on any page you want the contact form to display
